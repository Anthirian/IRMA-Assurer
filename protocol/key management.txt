Key		crypto type		Description
--------------------------------------------------------------------------------------------------------
pkS		assymmetric		Public key of Server. Used for encrypting messages to allow only Server to read.
skS		assymmetric		Secret key of Server. Used for decrypting received messages.
pkC		assymmetric		Public key of Client. Used for encrypting messages to allow only Client to read.
skC		assymmetric		Secret key of Client. Used for decrypting received messages.
Kts		symmetric		Session key generated by TLS 1.2. Ephemeral, so discarded after session.
